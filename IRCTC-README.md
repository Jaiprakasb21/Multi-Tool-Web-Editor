# IRCTC Booking Window Checker

A standalone web tool to check if your train journey date is within the IRCTC advance booking window. Get real-time booking status, countdown timers, and direct links to book tickets.

## Features

- ✅ **Real-time Booking Status** - Instantly check if your journey date is available for booking
- ⏰ **Countdown Timer** - See exactly when booking opens for unavailable dates
- 🎫 **Direct Booking Link** - One-click access to IRCTC website when booking is available
- 📅 **Smart Date Calculation** - Automatically calculates booking windows based on IST timezone
- 🔄 **Reset Functionality** - Easy reset button to clear all fields
- 📱 **Mobile Responsive** - Works perfectly on all devices

## How It Works

1. **Select Journey Date** - Choose any date you want to travel
2. **Check Status** - Click the button to check booking availability
3. **View Results** - See if booking is open, closed, or if the date has passed
4. **Book Tickets** - When available, the button transforms to "Book Tickets Now" for direct booking

## IRCTC Booking Rules

- **Advance Booking Period**: 60 days (excluding journey date)
- **Booking Opens**: Daily at 8:00 AM IST
- **Example**: On January 1st at 8 AM, tickets for March 2nd become available
